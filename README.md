# mfk_msu
Небольший скрипт, который парсит [сайт межфакультетских курсов (МФК) МГУ](new.mfk.msu.ru) и получает информацию о:
  - курсах
  - студентах
  - том, кто куда записался
## Запуск
''' python run.py
